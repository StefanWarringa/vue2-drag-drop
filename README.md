# Vue drag and drop

A simple trello-like board to explore the HTML5 drag and drop API written in Vue2.

### Project setup
```
npm install
```

#### Compile and hot-reload for development
```
npm run serve
```

#### Compile and minify for production
```
npm run build
```

#### Run your tests
```
npm run test
```

#### Lint and fix files
```
npm run lint
```
